## ✅ Image Format Decision Guide

| Use Case | Format |
|----------|--------|
| Photographs | JPEG or WebP |
| Logos with transparency | PNG or SVG |
| Icons | SVG |
| Simple animations | GIF |
| Modern performance | WebP |

## 🔗 Quick Reference

| Element | Purpose | Example |
|---------|---------|---------|
| `<a href="">` | Link | `<a href="page.html">` |
| `<a target="_blank">` | New tab | `target="_blank" rel="noopener"` |
| `<a href="#id">` | Section link | `<a href="#section">` |
| `<img src="">` | Image | `<img src="photo.jpg">` |
| `<img alt="">` | Alt text | `alt="description"` |
| `<picture>` | Art direction | See MDN |
| `./` | Current folder | `./image.jpg` |
| `../` | Parent folder | `../image.jpg` |
| `/` | Root | `/images/image.jpg` |
