## ✅ Setup Commands Cheatsheet

### Check Installation
```bash
node --version   # Should show v18+ or v20+
npm --version    # Should show v9+ or v10+
```

### Create React Project
```bash
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev
```

### VS Code Launch from Terminal
```bash
code .
```

## 🐛 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| `'node' is not recognized` | Node.js not installed or PATH not set |
| `'npm' is not recognized` | Reinstall Node.js |
| `Module not found` | Run `npm install` |
| `Port 5173 already in use` | Run `npm run dev -- --port 3000` |
| Blank white page | Check console for errors |
| JSX not working | File must be `.jsx` not `.js` |
| Styles not loading | Check CSS import path |
