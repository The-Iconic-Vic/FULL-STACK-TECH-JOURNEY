## ✅ CSS Methods Comparison

| Method | Pros | Cons |
|--------|------|------|
| Inline | Highest specificity, quick testing | Hard to maintain, no reuse |
| Internal | No extra HTTP requests | Single page only |
| External | Reusable, cached, clean separation | Additional HTTP request |

## 📝 Selector Specificity Quick Reference

| Selector | Example | Score |
|----------|---------|-------|
| Inline | `style=""` | 1000 |
| ID | `#id` | 100 |
| Class | `.class` | 10 |
| Element | `p` | 1 |

## 🎨 Color Formats Quick Reference

| Format | Example | Use Case |
|--------|---------|----------|
| Named | `red` | Quick prototyping |
| Hex | `#FF0000` | Most common, precise |
| RGB | `rgb(255,0,0)` | When opacity needed |
| HSL | `hsl(0,100%,50%)` | Intuitive color adjustments |
