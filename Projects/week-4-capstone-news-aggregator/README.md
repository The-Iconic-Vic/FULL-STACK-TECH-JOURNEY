# News Aggregator - Week 4 Capstone

## Project Overview
A complete news aggregator app using GNews API with category filtering, country selection, search, and save for later functionality.

## Technologies Used
| Technology | Purpose |
|------------|---------|
| HTML5 | Semantic structure |
| CSS3 | Styling, grid layout, responsive design |
| JavaScript | API calls, DOM manipulation, localStorage |
| GNews API | News data |

## Features

### Core Features ✅
| Feature | Description |
|---------|-------------|
| Search news | Search by keyword |
| Category filters | Technology, Sports, Business, etc. |
| Country-specific | News from different countries |
| Display headlines | Title, image, source, date |
| Read full article | Opens in new tab |
| Save articles | localStorage for later reading |
| Loading states | Visual feedback |
| Error handling | User-friendly messages |
| Responsive design | Mobile + desktop |

### Advanced Features ✅
- Saved articles section
- Remove from saved
- Category filtering
- Country selection

## API Used
| API | Endpoint | Purpose |
|-----|----------|---------|
| GNews | `/top-headlines` | Category-based news |
| GNews | `/search` | Search news |

## Setup Instructions
1. No API key required for GNews (free tier with limits)
2. Open index.html in browser
3. For production, consider signing up for a free API key at [GNews](https://gnews.io)
