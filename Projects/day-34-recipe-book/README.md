# Recipe Book Manager - Day 34 Project

## Features
- View all recipes as cards
- Add new recipes (title, ingredients, instructions)
- Edit existing recipes
- Delete recipes
- Mark recipes as favorites
- Search recipes by title
- Complex state management with arrays and objects

## Key React Concepts
- Array state updates: spread, map, filter
- Object state updates: spread operator
- Form handling with complex data
- Conditional rendering for edit mode
- Immutability in state updates
