# Business Card - Day 4 Project

## Project Overview
A styled digital business card demonstrating CSS basics including colors, typography, and layout.

## Skills Practiced
- External CSS linking
- Element, class selectors
- Named, hex, and gradient colors
- Font properties: family, size, weight
- Text properties: align, decoration, transform
- Box styling: border-radius, box-shadow, padding, margin
- Hover effects and transitions
