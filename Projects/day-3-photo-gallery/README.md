# 📸 Photo Gallery - Day 3 Project

## Project Overview
A multi-page photo gallery demonstrating links, images, and file paths.

## Skills Practiced
- Absolute and relative URLs
- `target="_blank"` with `rel="noopener noreferrer"`
- Thumbnail images linking to full-size versions
- Document-relative paths (`../`)
- Proper file structure organization
