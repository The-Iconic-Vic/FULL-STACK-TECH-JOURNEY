Promise vs Async/Await

Promise Version (.then/.catch)


fetchData()
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error(error);
  });

Async/Await Version


async function loadData() {
  try {
    const data = await fetchData();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}

What's happening?

  • async - Declares function returns a Promise
  • await - Pauses execution until Promise resolves
  • try/catch - Handles errors like synchronous code
  • 70% success rate - Simulates real API behavior
