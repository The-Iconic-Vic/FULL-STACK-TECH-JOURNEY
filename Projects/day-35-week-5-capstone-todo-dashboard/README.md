# Interactive Todo Dashboard - Week 5 Capstone

## Features
- Add, edit, delete todos
- Mark todos as complete
- Filter (All, Active, Completed)
- Sort by date, priority, title
- Search by title/description
- Bulk delete
- Dark/Light theme toggle
- LocalStorage persistence
- Export/Import JSON
- Statistics dashboard
- Progress bar
- Priority color coding
- Due date indicators

## Component Structure
App
├── Header
├── StatsDashboard
├── TodoForm
├── FilterBar
├── TodoList
│ └── TodoItem
└── ExportImport


## Setup Instructions

cd week-5-capstone-todo-dashboard
npm install
npm run dev
