# Weather Dashboard - Week 4 Capstone

## Project Overview
A complete weather dashboard application using OpenWeatherMap API with current conditions, 5-day forecast, recent searches, and dynamic backgrounds.

## Technologies Used
| Technology | Purpose |
|------------|---------|
| HTML5 | Semantic structure |
| CSS3 | Styling, animations, responsive design |
| JavaScript | API calls, DOM manipulation, localStorage |
| OpenWeatherMap API | Weather data |
| Geolocation API | User's location |

## Features

### Core Features ✅
| Feature | Description |
|---------|-------------|
| Search by city | Get weather for any city |
| Current weather | Temperature, condition, humidity, wind |
| High/Low temps | Daily min and max |
| Weather icons | Dynamic icons based on condition |
| 5-day forecast | Daily forecast with highs/lows |
| Dynamic background | Changes based on weather condition |
| Recent searches | Saved to localStorage |
| Loading states | Visual feedback |
| Error handling | User-friendly messages |
| Responsive design | Mobile + desktop |

### Advanced Features ✅
- Celsius/Fahrenheit toggle
- "Feels like" temperature
- Sunrise/Sunset times
- Geolocation (load weather for current location)

## API Used
| API | Endpoint | Purpose |
|-----|----------|---------|
| OpenWeatherMap | `/weather` | Current weather |
| OpenWeatherMap | `/forecast` | 5-day forecast |

## Setup Instructions
1. Sign up for a free API key at [OpenWeatherMap](https://openweathermap.org/api)
2. Replace `YOUR_API_KEY_HERE` in script.js with your API key
3. Open index.html in browser
