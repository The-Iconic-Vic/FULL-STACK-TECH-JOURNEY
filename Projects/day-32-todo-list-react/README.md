# Todo List App - Day 32 Project

## Features
- Add new todos
- Mark todos as complete/incomplete
- Edit existing todos
- Delete todos
- View statistics (total, active, completed, progress)
- Persists state in memory (component re-renders on changes)

## Key React Concepts
- useState hook for todos array
- Controlled component (input with onChange)
- Form submission (onSubmit + preventDefault)
- Event handling (onClick, onChange, onKeyPress)
- Array state updates (map, filter, spread)
- Conditional rendering
- Props drilling (parents to children)
