# Blog Preview Card - Day 8 Project

## Project Overview
A styled blog preview card demonstrating advanced CSS selectors, pseudo-classes, and pseudo-elements.

## Skills Practiced
- `:hover` pseudo-class with scale transform
- `::after` pseudo-element for arrow icon
- First paragraph special styling
- Image zoom effect on card hover
- Responsive design

## Advanced Selectors Used

| Selector | Purpose |
|----------|---------|
| `.card:hover` | Scale card and image on hover |
| `.read-more::after` | Arrow icon that moves on hover |
| `.card-preview` | Second paragraph with border-left styling |
| `::selection` | Custom text highlight color |
| `:first-child` / `:last-child` | Margin management |

## Key Features
- **Hover Scale:** Card scales up slightly on hover
- **Image Zoom:** Image zooms when card is hovered
- **Arrow Animation:** Arrow moves right on hover
- **Category Badge:** Styled category label
- **Meta Icons:** Calendar and clock icons using `::before`
- **First Paragraph:** Standard styling
- **Second Paragraph:** Special border-left styling for preview text
- **Custom Selection:** Purple highlight when text is selected
