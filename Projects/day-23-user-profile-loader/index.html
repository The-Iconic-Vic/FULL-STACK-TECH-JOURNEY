      Promise.all() Explained

      Promise.all() runs multiple promises simultaneously and waits for ALL to complete.

      • 📡 Fetching user profile: 1 second
      • 📝 Fetching user posts: 1.5 seconds
      • 👥 Fetching user friends: 1 second
      • Total time with Promise.all(): ~1.5 seconds (not 3.5 seconds!)
