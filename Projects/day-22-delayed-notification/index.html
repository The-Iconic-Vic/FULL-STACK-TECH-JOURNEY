🔔 Delayed Notification System

Simulating an API call with loading states and error handling

📡
Click the button to load data

How it works:

  1. Click "Load Data" button
  2. Shows loading state with spinner animation
  3. Simulates API call (2 second delay)
  4. 70% chance of success, 30% chance of error
  5. Displays success or error message accordingly

Promise States Demonstrated:

  • Pending - Loading message displayed
  • Fulfilled - Success message with ✓
  • Rejected - Error message with ❌
