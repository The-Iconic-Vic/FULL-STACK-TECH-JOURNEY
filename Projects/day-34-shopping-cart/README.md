# Shopping Cart - Day 34 Project

## Features
- View product list
- Add items to cart
- Update item quantities
- Remove items from cart
- Clear entire cart
- Calculate totals (subtotal, shipping, tax)
- Complex state updates with arrays and objects

## Key React Concepts
- Array state updates: spread, filter, map
- Object state updates: spread operator
- Immutability in state
- Nested state updates
